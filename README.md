**SyncTime** is a premium, open-source global meeting planner designed for distributed teams and international remote workers. It simplifies the headache of coordinating across multiple timezones with a professional, minimalist interface.

![SyncTime Logo](synctime_logo.png)

## ‚ú® Key Features

- **Apple-Style Time Pickers**: Intuitive 3D wheel pickers for smooth, precise time adjustments.
- **Searchable Global Database**: Powered by Select2, find over 400+ world cities and countries instantly (includes smart aliases like "Delhi" or "Beijing").
- **Live Sync**: All timezone cards update in real-time as you adjust one clock‚Äîall derived from a single UTC source of truth.
- **One-Click Scheduling**: Open a premium Oat UI modal to customize your meeting details and launch directly into **Google Meet** or **Microsoft Teams**.
- **Context-Aware Summary**: Automatically generates a meeting overview for your calendar invite, showing the exact time for every city on your list.
- **Premium Design**: Built with the Oat UI library for a minimalist, SaaS-like aesthetic with glassmorphism effects and professional typography.

## üöÄ Getting Started

Simply open `index.html` in any modern web browser. No compilation or installation is required.
